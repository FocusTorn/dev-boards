# Arduino Utilities

This directory contains utility scripts and tools for Arduino development.

## Structure

Similar to the Raspberry Pi setup, this directory can contain:
- Library search tools
- Template generators
- Build automation scripts
- Custom utilities

## Examples from Raspberry Pi Setup

The Raspberry Pi setup includes:
- `lib_search/` - Library search utilities
- `templates/` - Project templates

You can add similar utilities here as needed.
