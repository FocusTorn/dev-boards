## STATUS OVERVIEW
**PROTOCOL: Summarize project progress.**

1.  **Read Registry:** Read the **Tracks Registry**.
2.  **Parse Tracks:**
    -   Identify all registered tracks (Open, Closed, Archived).
    -   For active tracks, read their **Implementation Plan** (`plan.md`).
3.  **Summarize:**
    -   Calculate total phases and tasks across all active tracks.
    -   Count completed, in-progress, and pending tasks.
4.  **Report:**
    -   **Active Tracks:** List current work units.
    -   **Current Task:** Specific next step for the primary active track.
    -   **Progress:** overall tasks_completed/tasks_total (%).
