## CONTEXT LOADING
**PROTOCOL: Load track-specific documentation.**

1.  **Read Files:**
    -   Using the **Universal File Resolution Protocol**, resolve and read the **Specification** (`spec.md`) and **Implementation Plan** (`plan.md`).
    -   Resolve and read `NOTES.md` if it exists.
    -   Resolve and read the project **Workflow**.
2.  **Summarize:**
    -   Analyze `plan.md` to find the current phase and next task.
    -   Present a concise summary to the user:
        -   **Active Track:** <track_description>
        -   **Branch:** <current_branch>
        -   **Current Phase:** <phase_name>
        -   **Next Task:** <task_description>
        -   **Context:** <summary of progress or notes>
